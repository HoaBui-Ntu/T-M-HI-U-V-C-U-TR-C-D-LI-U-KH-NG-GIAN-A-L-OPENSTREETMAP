# TÌM HIỂU CẤU TRÚC KHÔNG GIAN ĐỊA LÝ OPENSTREETMAP
Dự án tìm hiểu cấu trúc không gian địa lý OpenStreetMap bao gồm:
Cấu trúc dữ liệu gốc OSM: node, way, relation.
Cấu trúc dữ liệu trong GIS: point, line, way.
Thuộc tính của các lớp dữ liệu và định dạng dữ liệu trên OSM và GIS.
Công cụ hỗ trợ nghiên cứu.
Phương pháp thu thập dữ liệu: Overpass API, Plugin QuickOSM của QGIS, tải bộ dữ liệu trên Geofabrik, sử dụng ngôn ngữ Python trên GG Colab.
Các bước tiền xử lý dữ liệu.
